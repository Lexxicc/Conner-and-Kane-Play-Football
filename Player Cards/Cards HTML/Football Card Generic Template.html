── HEADSHOT PREPARATION (REQUIRED before adding a player) ──
The portrait zone uses a stripe pattern background. The player photo MUST have a
transparent background so the stripes show around the silhouette. Solid-background
photos will render as a black square over the stripes.

canonical example : Headshot Lexxicc GK Player (Me).png (C. Johnson / GK 81)
css used : object-fit: cover; object-position: center 25%; — no blend mode

if source photo has a solid background, run background removal first:
python3 flood-fill from all 4 edges, threshold R/G/B <= 40 (catches near-black
studio backgrounds without touching dark navy kit). Script:

from PIL import Image
from collections import deque
img = Image.open("Headshot.png").convert("RGBA")
px = img.load()
w, h = img.size
def is_bg(p): return p[0] <= 40 and p[1] <= 40 and p[2] <= 40
queue = deque()
visited = [[False]*h for _ in range(w)]
for x in range(w): # seed top + bottom edges
for y in [0, h-1]:
if not visited[x][y] and is_bg(px[x,y]):
queue.append((x,y)); visited[x][y] = True
for y in range(h): # seed left + right edges
for x in [0, w-1]:
if not visited[x][y] and is_bg(px[x,y]):
queue.append((x,y)); visited[x][y] = True
while queue:
x, y = queue.popleft()
r,g,b,a = px[x,y]; px[x,y] = (r,g,b,0) # make transparent
for dx,dy in [(-1,0),(1,0),(0,-1),(0,1)]:
nx,ny = x+dx, y+dy
if 0<=nx<w and 0<=ny<h and not visited[nx][ny] and is_bg(px[nx,ny]):
visited[nx][ny]=True; queue.append((nx,ny))
img.save("Headshot Transparent.png")
